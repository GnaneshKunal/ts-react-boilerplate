### - [TSLint](https://palantir.github.io/tslint/)  :musical_note:

### API End points

Add your own API end points in server

### Usage
#### Install dependencies and DevDependencies
```
$ yarn install
```

#### Bundling
```
$ tsc -w // complies and -w (watches)
$ webpack -w // bundles and -w (watches)
```
#### Using the Server

```
$ yarn start
```
